# Contributing

Thanks for helping make sovereign, user-controlled Gen AI accessible to more people.

## Getting started

1. Fork the repository and clone your fork
2. Follow the **Development** section in [README.md](README.md)
3. Make your changes on a feature branch
4. Open a pull request with a clear description of what changed and why

## What we welcome

- Bug fixes and clearer error messages
- Documentation improvements (especially LLM provider setup)
- UI/UX polish that keeps the app simple for non-technical users
- Tests for backend logic (there is no test suite yet — adding one is valuable)

## Before you submit

- Keep changes focused — one concern per PR when possible
- Do not commit secrets, API keys, or personal chat data from `data/`
- If you change API behavior, update README or [docs/LLM_SETUP.md](docs/LLM_SETUP.md)

## Reporting issues

Include:

- OS and how you run the app (Docker vs dev)
- LLM provider, base URL, and model name (redact API keys)
- Steps to reproduce and any error text from the UI or server logs

## Code of conduct

Be respectful and constructive. This project exists to give people control over their AI tools — treat contributors the same way.
